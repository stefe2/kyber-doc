
# How to Adjust Component Positions

1. Open the original board image in an image viewer that shows cursor coordinates
   - macOS: Use Preview, hover to see coordinates in info panel
   - Windows: Use Paint, coordinates shown in status bar
   - Or use any image editor

2. Find the center point of each component:
   - SD Card slot: Look for the SD card holder
   - Speaker terminals: The screw terminals for speaker
   - Audio jack: The 3.5mm jack
   - Power input: Green screw terminal for power
   - I/O connections: Header pins along the edge
   - SBUS: Input pins for receiver
   - Maestro: Output pins for servo controller
   - ESP32: The main chip
   - DFPlayer: Small MP3 module

3. Update COMPONENT_POSITIONS in this script with correct coordinates

4. Run the script again to generate updated images
